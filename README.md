# NBA Simulation
golang in-memory

## To build and run
```
go build
./nbaSimulation
```

## To run with dynamic interval
```
TIME_INTERVAL_IN_SECONDS=5 go run main.go
```

# Docker
```
docker pull berkay94/nba-simulation:latest
```
